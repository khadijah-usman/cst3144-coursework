# CST3144 Lesson Booking App (Frontend)

This is the frontend single-page application (SPA) for my CST3144 full-stack coursework.
It is built with Vue 3 and Vite and allows users to browse, search and sort lessons,
add them to a cart, and complete a checkout form with validation.

## Frontend features

- Vue 3 single-page application using App.vue
- 10 lessons with subject, location, price and spaces
- Search and sorting by subject, location, price and spaces
- Add-to-cart with spaces decrement and restore
- Cart view with quantity updates and line totals
- Checkout form with validation for name (letters only) and phone (numbers only)
